# 网站上线前检查清单

## ✅ 必须完成项

### 1. 内容检查
- [ ] 替换微信二维码图片（images/wechat-qrcode.png）
- [ ] 检查所有文字内容是否准确
- [ ] 确认服务区域列表完整
- [ ] 验证案例描述无敏感信息
- [ ] 检查价格说明是否符合实际

### 2. 功能测试
- [ ] 在Chrome浏览器中打开index.html
- [ ] 点击所有导航链接，确认跳转正常
- [ ] 测试移动端菜单（缩小窗口或用手机）
- [ ] 滚动页面查看动画效果
- [ ] 检查二维码位置是否正确显示

### 3. 响应式测试
- [ ] 桌面端显示正常（1920px）
- [ ] 笔记本显示正常（1366px）
- [ ] 平板显示正常（768px）
- [ ] 手机显示正常（375px）
- [ ] 横屏竖屏切换正常

### 4. SEO检查
- [ ] Title标签包含核心关键词
- [ ] Meta description有吸引力
- [ ] H1标题唯一且合理
- [ ] 所有图片有alt属性
- [ ] 地域关键词覆盖全面

### 5. 性能检查
- [ ] 页面加载速度快（<2秒）
- [ ] 二维码图片大小合适（<100KB）
- [ ] 无控制台错误（F12查看）
- [ ] 无404资源加载失败
