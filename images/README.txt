# 图片存放说明

## 微信二维码图片

请将您的微信二维码图片放置在此文件夹中，命名为：
- **wechat-qrcode.png** (推荐) 或 wechat-qrcode.jpg

### 图片要求：
- 尺寸：建议 300x300 像素或更大
- 格式：PNG（透明背景）或 JPG
- 质量：清晰可扫描
- 文件名：必须为 `wechat-qrcode.png` 或修改HTML中的引用路径

### 如何获取微信二维码：
1. 打开微信 → 我 → 个人信息
2. 点击二维码名片
3. 保存到手机/电脑
4. 重命名为 wechat-qrcode.png
5. 放入此 images 文件夹

## 网站图标（可选）

如需自定义网站图标，请添加：
- **favicon.png** 或 favicon.ico
- 尺寸：16x16, 32x32, 或 64x64 像素

---

**注意：** 如果不添加二维码图片，页面会显示占位提示文字。
