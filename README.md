## 🚀 部署步骤

### 方法一：直接上传服务器（推荐）

1. **准备文件**
   - 确保 `images/wechat-qrcode.png` 已替换为您的真实微信二维码
   - （可选）添加 `images/favicon.png` 网站图标

2. **上传到服务器**
   ```bash
   # 使用FTP工具（如FileZilla）上传整个文件夹到网站根目录
   # 或使用SCP命令
   scp -r kouyi-website-suzhou-wuxi/* user@your-server:/var/www/html/
   ```

3. **访问测试**
   - 浏览器访问：`http://您的域名/`
   - 检查所有页面是否正常显示
   - 测试移动端适配

### 方法二：GitHub Pages免费托管

1. **创建GitHub仓库**
   - 新建公开仓库，如 `mechanical-interpreter`
   - 上传所有文件到仓库

2. **启用GitHub Pages**
   - Settings → Pages
   - Source选择：main branch / root
   - 保存后获得网址：`https://用户名.github.io/mechanical-interpreter/`

3. **自定义域名（可选）**
   - 在仓库Settings中添加自定义域名
   - DNS配置CNAME记录

### 方法三：Netlify/Vercel免费部署

1. **注册账号**
   - Netlify: https://netlify.com
   - Vercel: https://vercel.com

2. **拖拽部署**
   - 直接将文件夹拖拽到部署区域
   - 自动生成HTTPS网址

3. **绑定域名（可选）**
   - 在平台设置中添加自定义域名
